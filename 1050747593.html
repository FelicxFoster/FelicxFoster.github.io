дёҖзӣҙд»Ҙдёәж–җжіўйӮЈеҘ‘ж•°еҲ—жҡҙеҠӣйҖ’еҪ’зҡ„ж—¶й—ҙеӨҚжқӮеәҰжҳҜ$O(2^{n})$пјҢзҷҫеәҰдәҶжүҚеҸ‘зҺ°пјҢе®һйҷ…зҙ§з•Ңж—¶й—ҙеӨҚжқӮеәҰеә”иҜҘжҳҜ$O((\frac{1+\sqrt{5}}{2})^{n})$гҖӮдҪҶе…¶е®һ$O(2^{n})$еә”иҜҘд№ҹжІЎй”ҷпјҢжҜ•з«ҹжұӮж—¶й—ҙеӨҚжқӮеәҰжҳҜеҸӘиҰҒжұӮйҮҸзә§дёҚз®Ўзі»ж•°гҖӮ

иҜҒжҳҺ

зҙ§з•Ңж—¶й—ҙеӨҚжқӮеәҰиҜҒжҳҺ

ж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„и®Ўз®—иҝҮзЁӢеҫҲз®ҖеҚ•гҖӮе°ұжҳҜз®ҖеҚ•зҡ„$F(n) = F(n-1) + F(n-2)$
жҲ‘д»¬жҠҠи®Ўз®—$F(i)$жүҖйңҖзҡ„ж—¶й—ҙи®°дёә$T(i)$гҖӮз„¶еҗҺи®°и®Ўз®—еҠ жі•жүҖйңҖзҡ„ж—¶й—ҙдёә1пјҢйӮЈд№Ҳжҳҫз„¶жңү$T(n) = 1+ T(n-1) + T(n-2)$
иҝҷдёҖејҸеӯҗеҸҜд»ҘиҝӣиЎҢеҸҳеҪўгҖӮдёӨиҫ№еҗҢж—¶еҠ  1пјҢеҫ—еҲ°$T(n)+1 = (T(n-1)+1) + (T(n-2)+1)$
и®°$A(n) = T(n) + 1$пјҢжҳҫз„¶жңү$A(n) = A(n-1) + A(n-2)$
жҳҺжҳҫиҝҷжҳҜдёҖдёӘж–җжіўйӮЈеҘ‘ж•°еҲ—пјҢеҸӘдёҚиҝҮеҲқйЎ№жңүжүҖдёҚеҗҢ

дҪҶжҳҜжіЁж„ҸпјҢж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„еўһй•ҝзҺҮдёҺеҲқйЎ№жҳҜж— е…ізҡ„гҖӮиҜҒжҳҺеҰӮдёӢпјҡ
и®ҫжҹҗж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„еүҚдёӨйЎ№дёә$a, b$пјҢд»Ө$c = max(a, b)$пјҢжҳҫз„¶иҝҷдёӘж•°еҲ—зҡ„еўһй•ҝйҖҹеәҰдёҚдјҡи¶…иҝҮд»Ҙ$c, c$дёәеүҚдёӨйЎ№зҡ„ж–җжіўйӮЈеҘ‘ж•°еҲ—гҖӮиҖҢиҝҷдёӘж–җжіўйӮЈеҘ‘ж•°еҲ—е°ұжҳҜ$c, c, 2c, 3c, 5c, 8c, вҖҰ, F(n)c$пјҢеҚі$c(1, 1, 2, 3, 5, 8, вҖҰ, F(n))$
иҖҢ$c$жҳҜдёӘеёёж•°пјҢеӣ жӯӨж–җжіўйӮЈеҘ‘ж•°еҲ—ж— и®әеҲқйЎ№жҳҜеӨҡе°‘пјҢжёҗиҝӣеўһй•ҝзҺҮйғҪжҳҜзӣёеҗҢзҡ„гҖӮ
жүҖд»Ҙ$A(n)$зҡ„жёҗиҝӣеўһй•ҝзҺҮдёҺ$F(n)$зӣёеҗҢпјҢиҖҢ$T(n) = A(n) - 1$пјҢеўһй•ҝзҺҮд№ҹжҳҜдёҖж ·зҡ„

жүҖд»Ҙ$T(n) вҲҲ O(F(n))$
зҷҫеәҰеҸҜд»ҘзҹҘйҒ“ж–җжіўйӮЈеҘ‘ж•°еҲ—$F(n) = \frac{1}{\sqrt{5}}[(\frac{1+\sqrt{5}}{2})^{n}-(\frac{1-\sqrt{5}}{2})^{n}]$
жүҖд»Ҙ$T(n) вҲҲ O((\frac{1+\sqrt{5}}{2})^{n})$
